What is epi?

Epi is a programing language compiled in C++ useful for making console programs(Ej. Hello Worlds, etc...). You can use epi for reading files from internet, sending text to servers, creating dialogs, and much more.

Is epi easy?

Epi is easy for doing simple tasks, if you started programing on C++, Python or Java, this language would be easy for you.
