What's new on v_1.0?

  • Created this section for you to know new things of epi.
  • Added runIf function for running a function if the condition is 1 or other code if else.
  • Added img.imageToTxt function for reading an image and store it in a variable as text. (Java 21^ required)
  • Added netw.readFileFromInternet function for reading a file in internet. (Java 21^ required)
  • Added netw.readLineFileFromInternet function for reading a specified line from a file in internet. (Java 21^ required)
  • Created String Methods for modifing String content.
  • Created Visual Elements for making your program more colorful.
  • Added random function for creating random numbers with a max and a min.
  • Added Variable Functions for running variables with less text.
  • Added Epi examples in depth section in docs main screen for knowing cool things to do with epi.
  • Deprecated some functions
  • Added checkupdates command for when you run epi auto check for updates. (Java 21^ required) (Internet Connection required for correct working)
  • Created a gui for running the program with more options.
